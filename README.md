# Personal Finance Tracker

A Flask-based Personal Finance Tracker built during my internship.

## Features

- User Authentication
- Add Income & Expenses
- Budget Management
- Dashboard
- Reports (CSV & PDF)
- REST API
- Rate Limiting
- Docker Support
- Gunicorn Deployment

## Technologies

- Python
- Flask
- SQLAlchemy
- SQLite
- Docker
- GitHub

## Run

```bash
python run.py
```
